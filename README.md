# WifeAI

Личная ветка для работы с Cursor — отдельно от рабочих проектов.

## Как пользоваться

1. Открой этот репозиторий в Cursor.
2. Переключись на ветку `WifeAI` (или `cursor/wife-ai-d7b4`).
3. Напиши агенту, что хочешь сделать: сайт, текст, таблицу, идею — что угодно.
4. Смотри изменения в файлах и проси правки обычным языком.

## Папка `projects`

Сюда можно складывать отдельные задумки — каждая в своей подпапке.

```
projects/
  my-first-idea/   ← пример стартовой заметки
```

## Подсказки для первого запроса

- «Сделай простой сайт-визитку про …»
- «Помоги написать пост / письмо / список дел»
- «Разберись в этой таблице и сделай понятный отчёт»
- «Объясни, что изменилось в файлах»

Удачи — пиши, как думаешь, агент подстроится.
