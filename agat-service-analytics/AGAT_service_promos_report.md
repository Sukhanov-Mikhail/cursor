## 3. Рекомендации по стандартизации акций сети



### 3.1. Целевая архитектура акций (3 уровня)

1. **Федеральный must-have** (одинаково везде)
  - YouTube −10%  
  - Масло + диагностика 40 пунктов  
  - Купон −10% на следующее ТО (или явный эквивалент)
2. **Региональный шаблон** (выбирается из библиотеки)
  - «К жаре» (конденсатор/радиатор)  
  - «Первый визит −20/−25%»  
  - «Шиномонтаж + отель»  
  - «Вход 900 ₽»  
  - «Вечерний заезд»
3. **Бренд-специфика DЦ**
  - Hyundai/HAVAL/Jetour/Geely/Toyota — только при наличии бренда в городе



### 3.2. Конкретные решения


| Решение                      | Действие                                                                                                                | Приоритет |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------------- | --------- |
| A. Зафиксировать масло-сплит | Оставить 6500/7900 **только с обоснованием** (логистика/БК) или сблизить до одного коридора ±500 ₽                      | Высокий   |
| B. Дожать бедные витрины     | Владимир, Екатеринбург, Абакан, Сыктывкар: добавить минимум 1 входной оффер (900 ₽) + 1 acquisition (−20% первый визит) | Высокий   |
| C. Единый чек-лист витрины   | Обязательные 3 блока: якорь / сезон / вход. Запрет пустых карточек «Спецпредложение» без цены и срока                   | Высокий   |
| D. YouTube −10%              | Оставить федеральным; считать конверсию: просмотр → промокод → заказ-наряд по городу                                    | Высокий   |
| E. Сезонный шаблон           | «К жаре» сделать библиотечным пакетом с 2 ценовыми уровнями (10 / 15 тыс.)                                              | Средний   |
| F. Удобство                  | Пилот «Вечерний заезд» в 3–5 городах (по модели Минвод)                                                                 | Средний   |
| G. НН vs регионы             | Не упрощать НН до минимума: сохранить пакетность как эталон богатой витрины                                             | Средний   |




### 3.3. KPI на 90 дней

1. **100% городов** имеют полный федеральный слой (включая видимый купон ТО или эквивалент).
2. **0 городов** только с федеральным минимумом без локального усилителя.
3. Разница масла 6500/7900 либо объяснена политикой, либо сокращена.
4. Отчёт по YouTube −10%: CTR промокода и доля заказ-нарядов с промокодом по каждому городу.
5. Бедные витрины подняты минимум до **тира «средняя»**.
