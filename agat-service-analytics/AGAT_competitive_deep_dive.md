## 1. Executive summary

АГАТ сильнее конкурентов там, где есть **единый федеральный каркас акций** (YouTube −10%, масло+диагностика) **плюс** локальные пакеты.  
АГАТ проигрывает там, где:
1. витрина бедная при высокой конкуренции (**Екатеринбург**);
2. прямой бренд-оверлап с сильным локалом (**Абакан / Медведь**, **Ростов / Ключавто+Боравто**);
3. acquisition-скидки конкурента жёстче (**Волгоград / Арконт −40/−50%**);
4. масляный тариф **7 900** без усиления оффера (**Ростов, Саратов, Астрахань, Сыктывкар**).

**P0-города (действовать первыми):** Екатеринбург, Волгоград, Ростов, Абакан, Сыктывкар.

## 4. Позиция по городам (сжато)

### P0 — критично

**Екатеринбург**  
Конкуренция: Автомир, Автобан, Уникум, Lucky Motors, Fit — **5/5**.  
Витрина АГАТ: **бедная**.  
Риск: крупный рынок без сервис-амбиции = потеря постгарантии.  
Действие: вход 900 ₽ + −20% 1-й визит + сезон за 30 дней.

**Волгоград**  
Конкуренция: **Арконт** (−40/−50/899/ТО 5280) — эталон агрессии.  
У АГАТ: HAVAL −25% хорошо, но не закрывает весь легковой поток.  
Действие: общий «Давайте знакомиться −25/−30%» не только HAVAL.

**Ростов-на-Дону**  
Конкуренция: Ключавто + Боравто + АвтоРитэйл.  
У АГАТ: масло **7900** + средняя витрина.  
Действие: выровнять цену масла / добавить acquisition.

**Абакан**  
Конкуренция: **Медведь** с почти зеркальным китайским портфелем.  
У АГАТ: бедная витрина.  
Действие: сервисная война на постгарантии тех же марок.

**Сыктывкар**  
Конкуренция низкая, но витрина бедная и масло **7900**.  
Действие: быстрый фикс без дорогой медиа — высокий ROI.

### P1 — важно

| Город | Главный соперник | Фокус |
|---|---|---|
| Нижний Новгород | KOLOBOX / Автомир / Луидор | не отдать диагностику/бонусы независимым |
| Ставрополь | Боравто / Автопрестиж | пакеты поверх Renault-акции |
| Саратов | Лидер / Автомир | отдельно легковые и LCV |
| Минводы | Автопрестиж | масштабировать «вечерний заезд» |
| Астрахань | локалы + Fit | либо 6500, либо яркий пакет |

### P2 — укреплять

Красноярск, Ярославль, Иваново, Саранск, Киров, Владимир — держать шаблоны и подтянуть бедные (Владимир).

## 5. Strategic implications для АГАТ

### Что уже лучше рынка
1. **Федеральный YouTube −10%** — редкая единая механика (ближе только «−10% за подписку» у Луидора).  
2. Единый масляный пакет с диагностикой 40 пунктов — хорошая федеральная «полка».  
3. Локальные инновации (вечерний заезд, HAVAL −25%, сезонные 10–15к) — есть что масштабировать.

### Где отстаём
1. Acquisition-скидки первого визита (проигрыш Арконту).  
2. Age-программы для авто 3–5+ лет (проигрыш Автомиру).  
3. Funnel «диагностика 0 → ремонт со скидкой» (проигрыш KOLOBOX).  
4. Однородность витрин: 4 бедных города на фоне сильных конкурентов.

### Рекомендуемая модель ответа конкурентам

```
Федеральный слой (не трогать/считать):
  YouTube −10% + масло+диагностика + купон ТО

Библиотека противоядий:
  vs Арконт/локалы   → «1-й визит −20/−25%»
  vs Автомир         → age −15/−25/−30% на 3/4/5 лет
  vs KOLOBOX/Fit     → диагностика 0 + −15/−20% на ремонт 14–21 день
  vs Медведь/Боравто → бренд-постгарантийные пакеты тех же марок
  vs цена 7900       → выравнивание или компенсация пакетом
```

## 7. Ограничения

- Публичные витрины ≠ внутренняя доля рынка/маржа.  
- Рейтинг интенсивности — экспертная оценка на открытых данных.  
- Бренд-микросайты и офлайн-акции могут усиливать конкурентов сверх видимого.  
- Для точного share нужна внутренняя CRM/регистрации ТО/ mysterieshop.
